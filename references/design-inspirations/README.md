# Design Inspirations

Use this folder to collect visual references, screenshots, interface ideas, and future-stage inspiration for Olympus.

## Purpose

These references are not implementation tasks by default. They are a holding area for:

- dashboard layout ideas
- panel treatments
- navigation patterns
- visual hierarchy inspiration
- interaction ideas worth revisiting later

## How To Use This Folder

1. Save image files directly into this folder when you want to preserve them.
2. Add or update a matching markdown note describing:
   - what the reference is
   - what part of it is useful
   - what we might borrow later
   - what we should avoid copying literally
3. Keep references focused on decisions we may revisit, not random visual clutter.

## Naming Convention

- image files:
  - `YYYY-MM-DD-short-name.png`
  - `YYYY-MM-DD-short-name.jpg`
- notes:
  - `YYYY-MM-DD-short-name.md`
